Computer Setup

Students are required to provide their own laptops and to install free and open source software on those laptops.

R

Install R, RStudio and the tidyverse suite of R packages, as described here.

Git / GitHub

Register for a GitHub account and install git. See Happy Git With R for instructions on installing git and configuring things so git can communicate with RStudio.
